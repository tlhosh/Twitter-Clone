# Twitter Clone (X Clone)

## Project Overview

This project is a **Twitter (X) Clone** built using HTML, CSS, and JavaScript. It replicates the core features of a social media platform where users can post tweets, interact with posts, and view a timeline.

The goal of this project is to practice front-end development skills, including layout design, responsive UI, and dynamic interactions.

---

## Features

* Create and post tweets
* Like tweets
* Profile section with user details
* Responsive design for mobile and desktop
* Clean and modern UI inspired by Twitter (X)

---

## Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout 
* **JavaScript** – Functionality and interactivity
